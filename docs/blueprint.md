# **App Name**: Resume Rewriter

## Core Features:

- ATS Optimization Tool: Analyze resume content for Applicant Tracking System (ATS) compatibility, identify missing keywords, and make targeted recommendations to improve its score.
- Skill Suggestion Tool: Based on a job description or a user-provided skill, suggest relevant skills to include on the resume.
- Action Verb Enhancement: Suggest stronger action verbs to start bullet points based on the context of the experience, skills or accomplishments.
- Tone Adjustment: Revise the tone of the resume, adjusting it between formal and casual based on user preference.
- Content Formatting: Format resume sections like Experience, Education, and Skills to a visually appealing design with proper section titles, dates, and descriptions.
- Preview and Download: Allow users to preview the generated resume and download it in various formats.

## Style Guidelines:

- Primary color: Slate blue (#5F9EA0), to convey professionalism, stability, and trustworthiness.
- Background color: Light gray (#F0F0F0), for a clean, uncluttered backdrop that emphasizes readability.
- Accent color: Soft teal (#66CDAA), providing visual contrast and highlighting interactive elements.
- Font: 'Inter', a grotesque-style sans-serif, for both headlines and body text. Note: currently only Google Fonts are supported.
- Use a clean, structured layout with clear section headings, concise bullet points, and a balanced distribution of content for enhanced readability.
- Employ simple, professional icons to visually represent key skills and competencies, enriching the user experience and aiding information recall.
- Subtle transitions and animations will acknowledge the user's action whenever the text is modified, increasing the user's confidence.
